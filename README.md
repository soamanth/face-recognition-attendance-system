# Face Recognition Attendance System

## 📌 Project Description

The Face Recognition Attendance System is a Python-based application that automates student attendance using face recognition technology. The system identifies registered students from stored images and records attendance automatically in a CSV file.

This project eliminates manual attendance marking and demonstrates the practical use of Computer Vision and Python in real-world applications.

---

## 🚀 Features

- Face Detection
- Face Recognition
- Automatic Attendance Marking
- Student Record Management
- Attendance Storage in CSV Format
- Easy to Use Interface

---


---

## 📊 How It Works

1. Load registered student images.
2. Detect faces using OpenCV.
3. Compare detected faces with stored images.
4. Identify the student.
5. Mark attendance automatically.
6. Save attendance records in CSV format.
